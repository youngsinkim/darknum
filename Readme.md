#### 화면 구성






1. 내 정보를 먼저 가져온다. 
    -> 메뉴에 나의 기수 정보 내 정보에서 알 수 있다. 
    -> 즐겨 찾기 목록 검색 시에 나의 기수도 포함해서 검색해야 하기 때문에 

2. 과정별 기수 목록을 가져온다. 
    -> 과정별 기수 목록은 전체 보기에서 사용되지만 여기에 즐겨찾기 여부가 포함되어 있다. 
    -> 과정별 기수 목록을 db에 저장하고 즐겨찾기 설정된 것과 교수, 교직원, 내 기수를 포함하여 즐겨찾기 목록으로 구성한다. 
    
3. 즐겨찾기 업데이트 가져온다. 
    -> 로그인 시의 마지막 동기화 날짜를 넘겼을 경우, 즐겨찾기 목록이 있으면 즐겨찾기 목록을 가져와서 추가 및 업데이트 한다.
    -> 삭제된 것은 별도 remove로 내려온다. 
    -> 최초에는 동기화 날짜가 없으므로 0000-00-00 00:00 으로 넘긴다.
